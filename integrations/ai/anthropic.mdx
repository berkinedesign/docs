## Connect Anthropic Account
To be able to use your Anthropic API, follow these steps to setup integration.

<Steps>
  <Step title="Step 1">
    If you don’t have an Anthropic account, [sign up here](https://www.anthropic.com/) or contact their sales team for API access.
  </Step>
  <Step title="Step 2">
    Once you have access, navigate to your **Anthropic dashboard** page.
  </Step>
  <Step title="Step 3">
    Locate and copy your **API key**, next copy the generated API key. <br /> Make sure you have sufficient credits on your Anthropic account as they do not provide any free credits upon registration
    <Note>Keep your API key secure. It provides access to OpenAI’s services and should not be shared publicly.</Note>
  </Step>
  <Step title="Step 4">
    Login to Davinci AI using your admin credentials. 
  </Step>
  <Step title="Step 5">
    In **Admin Panel**, navigate to **AI Settings** -> **AI API Keys** tab. 
  </Step>
  <Step title="Step 6">
    Click on **Anthropic** tab there. 
    <img src="/images/ai/openai/anthropic-1.png" />
  </Step>
  <Step title="Step 7">
    Include your API key under **Anthropic API Key** input field. 
    <img src="/images/ai/openai/anthropic-2.png" />
  </Step>
  <Step title="Step 8">
    After you are done with configurations, click on **Save** button. 
  </Step>
</Steps>

<Check>
Congratulations! You have successfully comleted the Anthropic setup process.
</Check>
