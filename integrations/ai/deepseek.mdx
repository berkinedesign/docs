## Connect Deepseek Account
To be able to use your Deepseek API, follow these steps to setup integration.

<Steps>
  <Step title="Step 1">
    If you don’t have an Deepseek account, [sign up here](https://www.deepseek.com/en)
  </Step>
  <Step title="Step 2">
    Once logged in, navigate to **API Keys** page
  </Step>
  <Step title="Step 3">
    Generate a new API key by clicking **Create new API Key**, next copy the generated API key
    <Note>Keep your API key secure. It provides access to Deepseek’s services and should not be shared publicly.</Note>
  </Step>
  <Step title="Step 4">
    Login to Davinci AI using your admin credentials. 
  </Step>
  <Step title="Step 5">
    In **Admin Panel**, navigate to **AI Settings** -> **AI API Keys** tab. 
  </Step>
  <Step title="Step 6">
    Click on **Deepseek** tab there. 
    <img src="/images/ai/deepseek/deepseek-1.png" />
  </Step>
  <Step title="Step 7">
    Include your API key under **Deepseek API Key** input field. Base URL can remain as it is stated there.  
    <img src="/images/ai/deepseek/deepseek-2.png" />
  </Step>
  <Step title="Step 8">
    After you are done with configurations, click on **Save** button. 
  </Step>
</Steps>

<Check>
Congratulations! You have successfully comleted the Deepseek setup process.
</Check>
