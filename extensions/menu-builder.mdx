## Extension Configuration
To be able to use Menu Builder feature, follow these steps to setup Menu Builder Extension.

<Steps>
  <Step title="Step 1">
    Go to the **Marketplace** in **Admin Panel** and install **Menu Builder** extension after successfully purchasing it first.     
    <img src="/images/extensions/menu/menu-1.png" />    
  </Step>
  <Step title="Step 2">
    After installation is completed, go to **Menu Builder** tab in Admin Panel. 
    <img src="/images/extensions/menu/menu-2.png" />
  </Step>
  <Step title="Step 3">
    You now have completed the Menu Builder extension installation. Proceed to the section below on how to use it. 
  </Step>
</Steps>

## How to Use

Sidebar menu panel consists of two types of menu items, the ones that below to **User Panel** and to **Admin Panel**, users by default do not see any admin related menu items. 

- You can grab any menu item and grag and drop under/above any other menu item, or move it to be it's child element
<img src="/images/extensions/menu/menu-4.png" />
- Menu items that are available only for Admin groups are market with this special icon:
<img src="/images/extensions/menu/menu-3.png" />
- Menu items that are extensions, hence requires separatley instsallations first via the Marketplace are marked with this special icon:
<img src="/images/extensions/menu/menu-5.png" />
- If you turn off a menu item checkbox, then it will not be visible to anyone in the sidebar menu panel. 
- You can rename the menu item by simply clicking on the menu item name
- You can change the menu item icon by simply clicking on the menu item icon
- By clicking on the **Add Menu Item** button, you can generate new:
    - Group title
    - Divider line
    - New Menu item element
