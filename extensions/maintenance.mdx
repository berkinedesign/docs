## Extension Configuration
To be able to use Maintenance Mode feature, follow these steps to setup Maintenance Mode Extension.

<Steps>
  <Step title="Step 1">
    Go to the **Marketplace** in **Admin Panel** and install **Maintenance Mode** extension.     
    <img src="/images/extensions/maintenance/maintenance-1.png" />    
  </Step>
  <Step title="Step 2">
    After installation is completed, go to **AI Settings** page in Admin Panel and click on **Extensions** tab there. Click on **Maintenance Mode** there. 
    <img src="/images/extensions/maintenance/maintenance-2.png" />
  </Step>
   <Step title="Step 3">
    Configuration of Maintenance Mode is done on this page, there will be new menu item added in the Admin Panel. Below is the explanation of each field on Maintenance Mode page. 
    - To enable **Maintenance Mode**, turn on this checkboxes
    - **Maintenance Page Header** allows you to include main title of the message in large bold text
    - **Maintenance Banner** allows you to upload a user friendly banner as you prefer
    - **Maintenance Page Body Message** allows you write a short user friendly paragraph if needed which will be under the main page Header
    - **Maintenance Page Footer**, optional page footer details
    <img src="/images/extensions/maintenance/maintenance-3.png" />
  </Step>
  <Step title="Step 4">
    You know have completed the Maintenance Mode extension configuration.  
  </Step>
</Steps>
