## Extension Configuration
To be able to use Voice Isolator feature, follow these steps to setup this extension.

<Steps>
  <Step title="Step 1">
    Go to the **Marketplace** in **Admin Panel** and install **Voice Isolator** extension.     
    <img src="/images/extensions/voice-isolator/voice-1.png" />    
  </Step>
  <Step title="Step 2">
    After installation is completed, go to **AI Settings** page in Admin Panel and click on **Extensions** tab there. Click on **Voice Isolator** there. 
    <img src="/images/extensions/voice-isolator/voice-2.png" />
  </Step>
  <Step title="Step 3">
     Go to <a href="https://elevenlabs.io/" target="_blank">elevenlabs.io</a> website, sign up with them and get your **API Key** on your profile page after you successfully complete the registration process on their website.
    <Note>Store your API key securely and never share it publicly. The key provides direct access to your account.</Note>
  </Step>
   <Step title="Step 4">
   To control the settings of **Voice Isolator** exension, return to **AI Settings** page in Admin Panel and click on the **Extensions** tab, and select **Voice Isolator** extension. 
    1. Enter your **Elevenlabs API Key** in the input field. 
    2. Turn on the checkboxes for **Voice Isolator Feature** and **Voice Isolator Free Tier Access** 
    <Note>If you don't turn on Voice Isolator Free Tier Access, then users who did not subscribe to any subscription plan where you have Voice Isolator feature enabled, will not have any access to this feature</Note>
    3. After you are done with configuration, click the **Save** button.  
    <img src="/images/extensions/voice-isolator/voice-3.png" />
  </Step>
  <Step title="Step 5">
    You know have completed the Voice Isolator extension configuration. Proceed to the section below on how to use it. 
  </Step>
</Steps>

<Check>
You have successfully comleted the setup process of this extension.
</Check>

## How to Use
After you are done with the configuration of Voice Isolator feature, it will appear on the User panel there you can start using this extension.   

1. Go to Voice Isolator tab in the User panel 
2. Select your target audio file, you can upload following audio file extensions: **.mp3, .mp4, .ogg, .mpeg, .mpga, .m4a, .wav, .webm**, and uploaded audio file can be up to **500MB** in size
3. After you have selected your target audio file, click on the **Process** button and it will generate audio files stripping any background noise and audio for you as a final result.
